───────────────────────────────────────────────────────────────────────────
                         RULES FUNCTIONALITY
───────────────────────────────────────────────────────────────────────────

The following security rules were added for the 'calls' collection:

✅ CREATE: Users can create calls as themselves only
   - CallerId must match authenticated user
   - Cannot call yourself
   - Type must be 'voice' or 'video'
   - Status must be 'ringing' on creation

✅ READ: Only caller and receiver can read call documents
   - Third parties are blocked

✅ UPDATE: Only caller and receiver can update calls
   - Cannot change callerId or receiverId
   - Cannot change call type
   - Can update status, timestamps

❌ DELETE: No one can delete calls
   - Maintains audit trail
